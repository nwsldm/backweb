## Quick Start

Deploying your first application is simple. Just connect your Git repository and we'll handle the rest.

### Step 1: Sign Up

Create a free account at [deployhub.com/signup](/signup). You can sign up with GitHub or GitLab for faster onboarding.

### Step 2: Connect Your Repository

1. After signing in, click "New Project"
2. Select your Git provider (GitHub or GitLab)
3. Choose the repository you want to deploy
4. Configure deployment settings (or use defaults)

### Step 3: Deploy

We'll automatically build and deploy your application. The first deployment may take a minute, but subsequent deploys are instant.

### That's It!

Your application is now live on the DeployHub edge network. Every push to your repository will trigger a new deployment.
