## Adding Custom Domains

You can add custom domains to any deployment in just a few steps.

### Step 1: Add Domain

1. Go to your project settings
2. Click "Domains"
3. Enter your domain name (e.g., `www.yourdomain.com`)
4. Click "Add Domain"

### Step 2: Configure DNS

We'll provide you with DNS records to add to your domain registrar:

- **CNAME Record**: Point `www` to `cname.deployhub.io`
- **A Record**: Point root domain to our IP address

### Step 3: SSL Certificate

We automatically provision an SSL certificate using Let's Encrypt. This usually takes a few minutes.

### Wildcard Domains

For Pro and Team plans, you can add wildcard domains (e.g., `*.yourdomain.com`) to serve multiple subdomains from a single deployment.
