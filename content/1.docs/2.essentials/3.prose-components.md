## Getting Started with Serverless Functions

DeployHub supports serverless functions for building APIs alongside your frontend.

### Creating Functions

Create a `functions` directory in your project root:

```
your-project/
├── functions/
│   ├── api/
│   │   └── hello.js
└── ...
```

### Function Example

```javascript
// functions/api/hello.js
export default async (req, res) => {
  return res.json({
    message: 'Hello from DeployHub Functions!'
  })
}
```

### Supported Runtimes

- **Node.js** (18.x, 20.x)
- **Python** (3.9, 3.10, 3.11)
- **Go** (1.21+)
- **Rust** (1.70+)

### Deployment

Functions are automatically deployed with your frontend. Each function gets a URL like:

```
https://your-app.deployhub.app/api/hello
```

### Limits

- **Hobby**: 1,000 invocations/day
- **Pro**: 100,000 invocations/day
- **Team**: 1,000,000 invocations/day

### Environment Variables

Functions have access to the same environment variables as your frontend.
